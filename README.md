# Controle de Bar

Controle de Bar é uma aplicação console construída durante a **9ª edição**, da [Academia do Programador](https://www.academiadoprogramador.net "Site da Academia do Programador").

A aplicação consiste no cadastro de mesas, garçons, produtos e controle de contas.

## Funcionalidades

### Cadastro de Mesas  

	Requisito 1.1: Registro de Mesas
	Requisito 1.2: Visualização de Mesas registradas
	Requisito 1.3: Edição de Mesas já registradas
	Requisito 1.4: Exclusão de Mesas já registradas

### Cadastro de Garçons  

	Requisito 2.1: Registro de Garçons
	Requisito 2.2: Visualização de Garçons registrados
	Requisito 2.3: Edição de Garçons já registrados
	Requisito 2.4: Exclusão de Garçons já registrados

### Cadastro de Produtos  

	Requisito 3.1: Registro de Produtos
	Requisito 3.2: Visualização de Produtos registrados
	Requisito 3.3: Edição de Produtos já registrados
	Requisito 3.4: Exclusão de Produtos já registrados
	
### Controle de Contas  

	Requisito 4.1: Abertura de Novas Contas
	Requisito 4.2: Registro de Pedidos
	Requisito 4.3: Fechamento de Contas
	Requisito 4.4: Visualização de Contas em Aberto
	Requisito 4.5: Visualização Faturamento Diário	

# Tecnologias e Paradigmas utilizadas

- .net core
- Linguagem C#
- Pilares da Orientação a Objetos

# Como executar o projeto

### Pré-requisitos: dotnet 6.0

``` 
# clonar repositório
git clone https://github.com/alexandre-rech-lages/ControleDeBar-2023

# entrar na pasta do projeto
cd ControleDeBar.ConsoleApp

# compilar o projeto
dotnet build

# executar o projeto
dotnet run
```
